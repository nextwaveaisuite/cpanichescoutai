# CPA Niche Scout AI 2.0

Professional CPA niche research tool with AI-powered keyword analysis, CPA offer matching, domain finder, and website blueprints.

## Features

- **Keyword Analysis**: Score keywords by profit potential, difficulty, and traffic
- **CPA Offers**: Match with major CPA networks automatically
- **Domain Finder**: Find aged domains with traffic history
- **Blueprint Generator**: Get complete website structure recommendations
- **Email Capture**: Collect visitor emails with popup
- **Stripe Ready**: Accept payments for credit packages

## Pricing

- **Professional**: $19.00 - 50 Credits
- **Enterprise**: $29.00 - 100 Credits

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`
